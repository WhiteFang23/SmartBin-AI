## Project Overview
SmartBin AI is an AI-powered conversational assistant designed to help users
correctly segregate household waste. The project supports sustainable waste
management by providing real-time guidance, environmental awareness, and
responsible AI usage.

## SDG Alignment
- SDG 12: Responsible Consumption and Production
- SDG 11: Sustainable Cities and Communities
- SDG 13: Climate Action

## Features
- AI-based waste identification
- RAG-powered disposal guidance
- Environmental awareness prompts
- Responsible AI considerations

## Technologies & Concepts
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Conversational AI
- Sustainability & SDGs
